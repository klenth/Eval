# Node types for Eval:

## PrintArguments:
    StringArgument
    Expression

## Expressions (has a value):
    Add, Subtract, Multiply, Divide, Power
    Negate
    Literal (a value directly written in code)
    SquareRoot
    VariableAccess (read value of a variable)
    Input

## Statements (is a line of code):
    Print
    Assignment
